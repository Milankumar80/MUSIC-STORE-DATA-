# MUSIC-STORE-DATA-
DATA ANALYSIS OF A MUSIC STORE DATA USING SQL 
Project Overview:
This project involves analyzing the data of a music store to gain valuable insights into its operations, sales, and customer preferences. By utilizing SQL queries, we aim to answer various business-related questions that can help in making informed decisions to enhance the store's performance and customer satisfaction.

Dataset Description:
The dataset consists of multiple tables capturing different aspects of the music store's operations, including customers, orders, tracks, artists, albums, and genres. The key tables in the dataset are:

Customers: Information about the store's customers.
Orders: Details of the transactions made by customers.
Tracks: Information about the individual tracks available in the store.
Artists: Data about the artists whose tracks are sold in the store.
Albums: Information about the albums containing the tracks.
Genres: Categories of music tracks.
